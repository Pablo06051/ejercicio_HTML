GLOSARIO DE PROGRAMACION

N

Nombre de dominio: Dirección única que identifica un sitio web en Internet.

Ñ

Ñandú: Ave nativa de América del Sur, también conocida como Rhea.

O

Objeto: Instancia de una clase que contiene datos y métodos relacionados.

P

Polimorfismo: Capacidad de un objeto para tomar muchas formas, permitiendo que diferentes clases respondan a la misma llamada de método.

Q

Query: Consulta o pregunta realizada a una base de datos para obtener información específica.

R

Recursión: Técnica de programación donde una función se llama a sí misma para resolver un problema.

S

Sintaxis: Conjunto de reglas que definen la estructura y forma correcta de escribir instrucciones en un lenguaje de programación.

T

Tipo de dato: Clasificación que define el tipo de valor que puede almacenar una variable (ej. entero, cadena, booleano).

U

Unidad de medida: Estándar utilizado para cuantificar una magnitud física (ej. metro, kilogramo).

V

Variable: Espacio en memoria que almacena un valor y cuyo contenido puede cambiar durante la ejecución del programa.

W

Web: Conjunto de páginas y recursos interconectados a través de Internet.

X

XHTML: Lenguaje de marcado que combina HTML con XML, utilizado para estructurar contenido web.

Y

YAML: Lenguaje de serialización de datos legible por humanos, utilizado para la configuración de aplicaciones.

Z

Zig: Lenguaje de programación de sistemas diseñado para ser seguro, rápido y fácil de usar.
