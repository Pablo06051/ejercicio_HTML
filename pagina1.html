GLOSARIO DE PROGRAMACION

A

Algoritmo: Conjunto de instrucciones o reglas definidas para resolver un problema específico.

B

Bug: Error o fallo en un programa informático que provoca un comportamiento inesperado.

C

Compilador: Programa que traduce el código fuente de un lenguaje de programación a código máquina.

D

Depuración: Proceso de identificar y corregir errores en un programa informático.

E

Encapsulamiento: Técnica de programación que agrupa datos y funciones en una sola unidad.

F

Framework: Estructura de soporte que facilita el desarrollo de aplicaciones al proporcionar herramientas y bibliotecas.

G

Git: Sistema de control de versiones distribuido que permite gestionar cambios en el código fuente.

H

HTML: Lenguaje de marcado utilizado para estructurar contenido en la web.

I

IDE: Entorno de desarrollo integrado que proporciona herramientas para escribir, depurar y compilar código.

J

JavaScript: Lenguaje de programación utilizado principalmente para el desarrollo web del lado del cliente.

K

Kernel: Núcleo del sistema operativo que gestiona los recursos del hardware y permite la comunicación entre software y hardware.

L

Lenguaje de programación: Conjunto de reglas y sintaxis que permiten escribir instrucciones para que una computadora realice tareas específicas.

M

Modelo-Vista-Controlador (MVC): Patrón de diseño que separa la lógica de negocio, la presentación y el control de flujo en una aplicación.
